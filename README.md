# TicketLah

[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen)](https://aaronagai.github.io/ticketlah/)

Malaysia's homegrown event platform — discover and host events across the country.

**Live site:** [https://aaronagai.github.io/ticketlah/](https://aaronagai.github.io/ticketlah/)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Event discovery with city and category filters
- Malaysian-themed design (Jalur Gemilang colours, local copy)
- Event detail pages with booking CTA
- Create event form (UI prototype)
- Mock data for KL, Penang, JB, Ipoh, KK, and Melaka events

## Stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- TypeScript
